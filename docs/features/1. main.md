Feature Specification
========================

Das Dokument beschreibt den Grundaufbau der App.

Es ist in verschiedene Abschnitte unterteilt, die jeweils eine bestimmte Funktionalität oder ein Feature der App beschreiben.

1. App-Start
-----------------

Beim Start der App wird der Benutzer auf die Setup-Seite geleitet, wo er grundlegende Einstellungen vornehmen kann. Dies ist das hinterlegen der URL des RomM Servers, sowie Benutzername und Passwort. Diese Informationen sind notwendig, um eine Verbindung zum Server herzustellen und die App korrekt zu konfigurieren. Diese Daten werden lokal gespeichert, um eine erneute Eingabe bei zukünftigen Starts zu vermeiden. Sollte dabei ein Token generiert werden, wird dieser ebenfalls lokal gespeichert. Das Passwort soll nicht lokal gespeichert werden, um die Sicherheit der Benutzerdaten zu gewährleisten. Nach Abschluss des Setups wird der Benutzer zur Hauptseite der App weitergeleitet. Das speichern soll in einem verschlüsselten Format erfolgen, um die Sicherheit der Daten zu gewährleisten. Die implementierung kann dabei später ausgetauscht werden.

2. Hauptseite
-----------------

Die Hauptseite besteht aus einer Tab-Leiste, die es dem Benutzer ermöglicht, zwischen verschiedenen Ansichten zu wechseln. Die Tabs sind wie folgt: Plattforms, Collection, Suche, Profil.
